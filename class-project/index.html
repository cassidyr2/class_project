The Basic Language Of the Web: HTML

Posted By Mr McCoy on Wednesday, May 19th 2030

All websites and web applications are built using three fundamentals technologies: HTML, CSS and Javascript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it

What is HTML?

HTML stand for Hyper Text Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, video, etc. web browser understands HTML will render HTML codes as websites.

In HTML, each element is made up of 3 parts:

  1. The Opening tag
  2. The Closing tag
  3. The Actual element
